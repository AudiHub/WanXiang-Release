📖 使用文档

🔗 添加影视源

打开设置 → 源管理 → 点击右上角 + 号。支持两种模式：

  • 手动添加：输入源名称和 API 地址（Apple CMS / TVBOX 格式），保存后自动加载
  • 导入/导出：可将现有源列表导出为 JSON 文件，跨设备迁移或备份

💡 添加多个源可实现并行搜索，某一源失效时自动从其他源获取结果

📺 电视直播

添加电视源后在顶部切换到「电视」标签：

  • 支持 M3U / TXT 格式直播源，频道自动分组管理
  • EPG 节目指南：点击节目单按钮查看当前频道完整节目表
  • 频道切换：左右滑动或使用上一频道/下一频道按钮
  • 播放器底部支持频道列表、节目单、切换源等快捷操作

🎵 音乐模块

顶部切换到「音乐」标签，首次使用需导入落雪音乐音源脚本：

  • 导入音源：设置 → 音源插件 → 导入落雪音乐 .js 音源脚本
  • 搜索播放：支持单曲搜索、歌单搜索、排行榜浏览
  • 歌词：全屏歌词界面，支持逐字同步、翻译显示
  • 播放队列：顺序/随机/单曲循环，支持收藏管理
  • 多平台聚合：咪咕/酷狗/酷我/网易云/QQ音乐，一键切换搜索

🔍 搜索与筛选

  • 拼音搜索：支持全拼、首字母，输入拼音自动匹配中文片名
  • 多源并行：启用多个源时，搜索结果来自所有可用源，去重合并
  • 分类浏览：首页按类型分类滚动，支持悬停/点击进入分类列表

🎮 播放控制

  • 手势控制：点击显示/隐藏控制栏；桌面端双击进入全屏
  • 倍速播放：底部倍速按钮支持 0.5x ~ 2.0x
  • 音量调节：桌面端水平滑块直接调节；手机端点击音量图标弹出竖向滑块
  • 选集切换：手机端底部「选集」按钮打开右侧集数面板；桌面端详情面板直接显示
  • 进度恢复：观看记录自动保存，再次打开从上次位置继续播放

🎨 主题与外观

  • 明暗主题：自动跟随系统，也可手动切换。播放器界面采用毛玻璃半透明风格
  • 青少年模式：设置 → 青少年模式 → 设置密码 → 可选择隐藏特定影视源，为儿童提供安全观影环境

🔄 更新升级

应用启动后自动检测 GitHub Releases 最新版本，检测到新版本后弹出下载提示。也可在设置中手动检查更新。Windows 用户推荐使用安装器版本（自动创建桌面快捷方式 + 卸载入口）。

⚙️ 其他设置

  • 数据管理：观看历史、收藏、源配置本地存储，可一键清除
  • 移动端：横竖屏自动适应布局；电视模式底部按钮囊括所有操作入口
